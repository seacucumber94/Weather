How to make your own language file:

To explain we assume you want to make a Italian language file.
1.  make a copy of wulanguage-en.txt   
  and 
    name it wulanguage-it.txt  (it = language code for Italian)

2.  Open wulanguage-it.txt in a program editor or in notepad++
    Do NOT use MS-word as a word processing program 
    will add all kind of strange characters	

3.  Replace the right part of every line with the correct translation.	

Example change
langlookup|wsreport_Wind|Wind|
to
langlookup|wsreport_Wind|Vento|

3.  Do NOT touch or change the first two parts, 
       even if you see a typing error there.
    LEAVE as is

You can now run the program and the standard texts are translated to your language when adding the lang=it (in this example) to the browser address
